2D Game - AA Replica

## Description ##
This game is a replica of a mobile AA game. The objective is to get as many pins in the circle as possible.

## Features ##

* Multiple Scenes
* Score tracking
* Adjustable Settings

## Changes ##
* Name of project changed from “Assignment02” to “Assignment04”
* Replaced old video demonstration with a new one
* Added instructions on how to play on the intro scene
* Added High Score to the end scene.
* Added the Homer Simpson “D’oh!” sound when the player loses
* Players can now put name in on the intro screen and it will display during gameplay
* More information has been added to the main level, such as games played, pin speed, and rotator speed
